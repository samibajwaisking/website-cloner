## Example 1 — samioutic.com

**File:** `samioutic-clone.html`  
**Source:** https://samioutic.com  
**Type:** Personal portfolio / AI educator site

### What was cloned:
- Dark purple theme with gradient accents
- Sticky navbar with mobile hamburger
- Hero with typewriter animation + animated stat counters
- About section with photo, tags, and timeline
- 8 credential/certification cards
- 17 service cards with hover effects
- Free demo modal with WhatsApp integration
- 20-photo gallery with lightbox
- Contact section with social links + message form
- Footer

### Features demonstrated:
- CSS custom properties for easy rebranding
- Scroll reveal animations
- Counter animations on scroll
- Fully mobile responsive
- Zero external dependencies (single HTML file)
