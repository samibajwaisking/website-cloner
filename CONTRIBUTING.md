# Contributing to Website Cloner

Thank you for your interest in contributing! Before you do anything, please read this document fully.

## ⚠️ Pre-Contribution Requirements

You CANNOT contribute without completing ALL of these first:

- [ ] ⭐ **Star this repository** — [click here](https://github.com/samibajwaisking/website-cloner)
- [ ] 👤 **Follow @samibajwaisking** on GitHub — [click here](https://github.com/samibajwaisking)
- [ ] 📖 **Read the [LICENSE](LICENSE) in full**
- [ ] ✅ **Agree to the Custom Attribution License terms**

If you skip any of these, your pull request will be rejected without review.

---

## What You Can Contribute

- Bug fixes in SKILL.md instructions
- New edge case handling examples
- Additional example clone outputs (with your credit + Sami's credit both)
- Documentation improvements
- Translation of README (must keep all attribution requirements)

## What You Cannot Contribute

- Removing or weakening attribution requirements
- Changing the license
- Removing credit to Sami Bajwa from any file
- Adding competing author names as primary credit

---

## Pull Request Template

Your PR description **must** include:

```
## What I changed
[describe your changes]

## Confirmation
- [ ] I have starred this repository
- [ ] I am following @samibajwaisking on GitHub
- [ ] I agree to the Custom Attribution License
- [ ] I have not removed any credit to Sami Bajwa
- [ ] I understand my contribution will be licensed under CAL-1.0-SamiBajwa
```

PRs without this template will be closed immediately.
