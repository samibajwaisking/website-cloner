# Changelog

All notable changes to Website Cloner Skill are documented here.

---

## [2.0.0] — Latest

### Major Update — Engine Upgrade

**Added**
- GSAP animation support — cloned sites now include smooth scroll animations,
  entrance effects, and hover transitions
- Tailwind CSS integration — output uses utility-first classes for faster,
  cleaner responsive layouts
- Improved component detection — Claude now identifies and recreates navbars,
  hero sections, feature grids, testimonials, pricing tables, and footers more reliably
- Better mobile responsiveness — all outputs are mobile-first by default

**Improved**
- More accurate color palette extraction from reference sites
- Better font matching and typography hierarchy
- Cleaner, more maintainable HTML structure in output files

---

## [1.0.0] — Initial Release

### First Public Release

**Core Skill**
- Website cloning via Claude AI — URL or screenshot to HTML/CSS/JS
- Vanilla JS output — no framework dependencies
- Self-contained single HTML file output
- Basic responsive layout support
- Custom Attribution License (CAL-1.0)

**Repository**
- README.md with full usage guide
- LICENSE with Star + Follow requirement
- CONTRIBUTING.md
- examples/ folder with sample outputs
