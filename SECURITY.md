# Security Policy

## Scope

This repository contains a Claude AI Skill for cloning websites into clean HTML/CSS/JS files.
There is no executable backend, no server, and no user data collection.

Security concerns for this project include:

- Prompt injection patterns that could cause Claude to produce malicious HTML/JS output
- Instructions that bypass Claude's safety guidelines
- Skill content that could be used to clone websites for phishing or fraud
- License violations or unauthorized redistribution

---

## Responsible Use

This skill is intended for:
- Recreating your own websites in a different stack
- Learning from existing UI/UX patterns for personal study
- Building original projects inspired by design patterns

This skill must NOT be used for:
- Cloning websites to impersonate brands or commit fraud
- Scraping copyrighted content without permission
- Bypassing paywalls or authentication systems

---

If you discover content in this skill that could enable harm, fraud, or misuse:
